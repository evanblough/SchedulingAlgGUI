MainWindow::MainWindow This is the standard constructor for my main window widget. It populates the empty scroll areas in the main with the four child widgets of the app.

Parameters
parentThis is the parent widget. It is not used.
schedule_panelThis is the schedule panel widget. This is a graph generated by Qpainter.
workload_panelThis is the workload panel. This panel allows a user to specify the properties of the periodic and aperiodic workloads by supplying input files or manually inputting values.
analysis_panelThis is the analysis panel. There are GUI elements in this widget that allow a user to specify what types of properties they want to observe
display_adjusterThis panel is used for adjusting the display of the schedule_panel. It allows a user to zoom and specify time bounds to display.
controllerThis object controls all the widgets. It reads signals from widgets and routes the information to the other panels.
